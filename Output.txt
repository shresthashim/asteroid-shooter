🎮 CONTROLS:
   ⬆️  UP Arrow    - Thrust forward
   ⬇️  DOWN Arrow  - Reverse thrust (go backward)
   ⬅️  LEFT Arrow  - Rotate left
   ➡️  RIGHT Arrow - Rotate right
   🚀 SPACEBAR    - Shoot
   📊 R Key       - Show game report (after game over)
   🔄 S Key       - Start new game
   ➡️  N Key       - Next level (when available)

🎯 OBJECTIVES:
   💥 Destroy all asteroids to advance levels
   🌟 Survive as long as possible
   📈 Get the highest score

Press any key to start!
